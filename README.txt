CARA GANTI HEADER / LOGO:

1. File cover utama ada di:
   assets/header-cover.png

   Kalau mau ganti cover, siapkan gambar ukuran 1920 x 800 px,
   lalu rename menjadi header-cover.png dan timpa file lama.

2. File logo navbar/footer ada di:
   assets/logo-tfc.png

   Kalau mau ganti logo, siapkan logo PNG,
   lalu rename menjadi logo-tfc.png dan timpa file lama.

3. Ganti link Google Form:
   buka index.html,
   cari:
   https://forms.gle/contoh-link-1
   sampai:
   https://forms.gle/contoh-link-12

4. Ganti nomor WhatsApp:
   cari:
   https://wa.me/628xxxxxxxxxx

5. Setelah diedit, buka index.html di browser.
